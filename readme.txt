== Description ==

GL Color Palette Generator is a powerful WordPress plugin that leverages various AI providers to generate beautiful and harmonious color palettes for your designs. Whether you're a designer, developer, or content creator, this plugin helps you create professional color schemes with ease.

Theme variations are optional design configurations within a block theme that allow you to change the look of your website without switching to a different theme. This plugin generates 16 harmonious variations for you to choose from.

Features:

* Generate 16 AI-powered theme variations
* Preview variations in real-time within the Site Editor
* Switch variations easily without changing themes
* Generate color palettes using natural language descriptions
* Support for multiple AI providers:
  * OpenAI
  * Anthropic
  * Azure OpenAI
  * Google PaLM
  * Cohere
  * Hugging Face
* Easy-to-use interface
* Color harmony validation
* Export palettes in multiple formats

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/gl-color-palette-generator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure your preferred AI provider API keys in the plugin settings
4. Generate palettes in the admin area
5. To switch theme variations:
   * Go to Appearance > Editor in your WordPress dashboard
   * Under Design, click on Styles
   * Click on a variation in the left panel to preview it
   * Click "Save" to apply the variation to your site

== Frequently Asked Questions ==

= Which AI provider should I use? =

Each AI provider has its strengths. OpenAI and Anthropic generally provide the most consistent results, while other providers may offer different pricing or features. Try different providers to see which works best for your needs.

= Do I need API keys for all providers? =

No, you only need an API key for the provider(s) you want to use. You can configure one or multiple providers.

= How do I switch between different color palettes? =

Theme variations make it easy to change your site's appearance:
1. Go to Appearance > Editor in your WordPress dashboard
2. Under Design, click on Styles
3. Click any variation in the left panel to preview how it looks
4. When you find one you like, click "Save" to apply it to your site
